# Windsurf — wiring

**Global:** `~/.codeium/windsurf/memories/global_rules.md`, written by the sync script.
⚠️ Windsurf enforces a **~6000-character limit** on global rules — the script truncates and prints
a warning if `rules/AGENTS.md` exceeds it. Keep the global ruleset tight.

**Project:** Windsurf reads root `AGENTS.md` natively; the script also writes `.windsurf/rules/global.md`.

**Facts/recall:** via the Obsidian MCP server (`~/.codeium/windsurf/mcp_config.json` — see [[mcp-setup]]).
